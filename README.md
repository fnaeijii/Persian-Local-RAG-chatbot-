# Persian-Local-RAG-chatbot-
یک سیستم پرسش و پاسخ هوشمند (QA) مبتنی بر معماری RAG (Retrieval-Augmented Generation) با رابط کاربری بومی‌سازی شده‌ی فارسی. این برنامه به کاربران اجازه می‌دهد فایل‌های متنی یا PDF (مانند دفترچه‌های راهنمای تجهیزات، پروتکل‌های ایمنی یا مستندات سازمانی) را آپلود کرده و با استفاده از قدرت مدل‌های زبانی (مانند DeepSeek) سوالات خود را از داخل متن بپرسند.

## ✨ ویژگی‌های کلیدی
- **رابط کاربری کاربرپسند:** طراحی مدرن، راست‌چین (RTL) و بهینه‌سازی شده با فونت وزیرمتن.
- **پردازش اسناد:** پشتیبانی از فایل‌های `PDF` و `TXT` با استفاده از کتابخانه قدرتمند `PyMuPDF` برای استخراج دقیق متن.
- **موتور جستجوی معنایی:** ساخت ایندکس محلی برای بازیابی سریع‌تر و دقیق‌تر پاراگراف‌های مرتبط با سوال کاربر.
- **یکپارچگی با DeepSeek:** ارسال پاراگراف‌های استخراج‌شده به LLM برای تولید پاسخ‌های طبیعی و دقیق بر اساس متن آپلود شده.
- **سرور محلی سبک:** اجرای سریع با `http.server` پایتون و قابلیت دسترسی آسان در شبکه داخلی (LAN) برای استفاده تیمی.

## 🛠 تکنولوژی‌های استفاده شده
- **Backend:** Python (http.server, socketserver)
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Document Processing:** PyMuPDF (fitz)
- **AI / NLP:** یکپارچگی با مدل‌های زبانی (تنظیم شده در فایل `rag_app.py`)
